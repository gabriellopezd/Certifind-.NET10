# Certifind .NET10

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/gabriellopezd/Certifind-.NET10.git
   ```

2. **Install dependencies**:
   Navigate to the project directory and install the required packages:
   ```bash
   cd Certifind-.NET10
   dotnet restore
   ```

3. **Run the application**:
   ```bash
   dotnet run
   ```
